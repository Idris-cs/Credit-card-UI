# Credit Card UI

A modern and elegant credit card user interface built with HTML and CSS. This project showcases a realistic credit card design with responsive styling.

## Features

- **Clean Design**: Modern and professional credit card layout
- **Responsive**: Adapts to different screen sizes
- **Master Card Theme**: Styled as a Master Card credit card
- **Card Details**: Displays card number, cardholder name, and valid date
- **Visual Elements**: Includes card chip and logo imagery
- **Professional Styling**: Uses gradient backgrounds and smooth shadows

## Project Structure

```
Credit Card UI/
├── index.html      # Main HTML file with card structure
├── styles.css      # CSS styling for the card interface
├── images/         # Image assets
│   ├── card.png    # Card logo
│   ├── chip.png    # Card chip imagery
│   └── style-bg.jpg # Card background texture
└── README.md       # Project documentation
```

## Technologies Used

- **HTML5**: Semantic markup for card structure
- **CSS3**: Flexbox layout and styling
- **Images**: PNG and JPG for visual assets

## How to Use

1. Clone or download this project
2. Open `index.html` in your web browser
3. View the styled credit card interface

## Customization

You can easily customize the card by editing:

- **Card Number**: Modify the number in `index.html`
- **Cardholder Name**: Update the name value
- **Valid Date**: Change the expiration date
- **Colors & Styling**: Edit `styles.css` to change the appearance

## Browser Compatibility

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
